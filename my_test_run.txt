=== Питання ===
Що таке інкапсуляція в ООП?
=== Еталон ===
Інкапсуляція приховує внутрішній стан об'єкта і надає контрольований доступ через методи або властивості.
=== Відповідь ШІ ===
Інкапсуляція — це механізм, який дозволяє приховувати внутрішній стан об'єкта та надавати контрольований доступ до нього через методи або властивості. Це допомагає захистити дані від некоректного використання.
=== Питання ===
Напиши короткий приклад Python-функції, що повертає квадрат числа.
=== Еталон ===
def square(x):
    return x * x
=== Відповідь ШІ ===
def square(x):
    return x * x
=== Питання ===
Чим список Python відрізняється від кортежу?
=== Еталон ===
Список змінюваний, а кортеж незмінюваний; обидва зберігають впорядковані елементи.
=== Відповідь ШІ ===
Список є змінюваним (можна додавати, видаляти, змінювати елементи), а кортеж — незмінюваним (після створення його не можна змінити). Обидва зберігають впорядковані елементи.
